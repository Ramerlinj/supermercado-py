# 🛒 Market

Market is a modern **supermarket web application** focused on delivering a clean, intuitive, and user-friendly shopping experience. The project is designed as a digital grocery store where users can browse product categories, view best-selling items, and interact with products in a simple and efficient way.

This project was built as a practical application to combine **backend development with Flask and Python** alongside a carefully designed frontend focused on **UX/UI clarity**.

---

## 🚀 Features

* 🥦 Product categories (Fruits, Vegetables, Bakery, Dairy, Meat, Frozen Foods)
* ⭐ Best-selling products section
* 🏷️ Promotional offer badges
* ➕ Quick add-to-cart interaction
* 🎨 Clean, modern, and responsive UI
* 📦 Scalable structure for future features (auth, cart, orders)

---

## 🧠 Design Philosophy

Market prioritizes:

* **Visual hierarchy** to highlight prices and offers
* **Soft and natural colors** to reinforce freshness
* **Minimalist cards** for easy product scanning
* **Fast navigation** through category icons

The goal is to create a pleasant shopping experience without overwhelming the user.

## ⚙️ Installation & Setup

1. Clone the repository

```bash
git clone https://github.com/your-username/market.git
cd market
```

2. Create and activate a virtual environment

```bash
python -m venv venv
source venv/bin/activate  # Linux / Mac
venv\Scripts\activate     # Windows
```

3. Install dependencies

```bash
pip install -r requirements.txt
```

4. Run the application

```bash
flask run
```

5. Open your browser at

```
http://127.0.0.1:5000
```
